# Assets Instagram @nakesproid

Struktur folder buat nyimpen & ngatur semua konten visual IG NakesPro.

## 📁 Struktur Folder

```

├── templates/         ← Master template Canva (png/jpg preview aja)

│   ├── carousel-portofolio.png

│
├── screenshots/       ← Screenshot website client & app buat bahan konten
│   ├── calm-warm-hero.png
│   ├── modern-light-hero.png
│   ├── halaman-layanan.png
│   ├── halaman-payment.png

├── batch-1/           ← Hasil desain 4 postingan perdana (minggu 1)

│   ├── post2-portofolio/

│

```

## 🚀 Cara Pakai

1. **Siapin bahan** — screenshot website client, simpan di `screenshots/`
2. **Desain di Canva** — pake 3 master template (Carousel Edukasi, Carousel Portofolio, Story)
3. **Simpan hasil** — export .png, taruh di folder `batch-1/` atau `batch-2/` sesuai nomor post
4. **Posting** — ambil dari folder, upload ke IG
5. **Arsip** — pindahin ke `hasil-jadi/` setelah posting
