# NakesPro.id

Situs marketing jasa pembuatan website untuk tenaga kesehatan mandiri (perawat, bidan, fisioterapis, home care).

**Model bisnis:**  
- **Paket Custom** — jasa pembuatan website branded (one-off).
- **Paket Template** — website siap pakai multi-tenant + langganan.

Landing page ini menjual kedua paket, mengarahkan client ke app (`nakespro-app`) untuk register, bayar, dan isi form detail.

### Rujukan Tautan
- **PRD.md** — Deskripsi lengkap dan tujuan proyek.
- **MVP-SPEC.md** — Detail spesifikasi teknis untuk `nakespro-app`.

## Stack
- **Framework:** Next.js 16 (App Router) + React 19 + TypeScript
- **Styling:** Tailwind CSS v4
- **Icons:** lucide-react
- **Deploy:** VPS Easypanel (`askep_nakespro`)

## Development
```bash
npm install
npm run dev
```
