# NakesPro.id

Situs marketing jasa pembuatan website untuk tenaga kesehatan mandiri (perawat, bidan, fisioterapis, home care).

Model bisnis **hybrid**:
- **Paket Custom** — jasa pembuatan website branded (one-off).
- **Paket Template** — website siap pakai multi-tenant + langganan (fase berikutnya, repo terpisah).

Lihat [PRD.md](./PRD.md) untuk detail lengkap.

## Stack
- Next.js 16 (App Router) + React 19 + TypeScript
- Tailwind CSS v4
- Deploy: VPS Easypanel → nakespro.id

## Development
```bash
npm install
npm run dev
```
